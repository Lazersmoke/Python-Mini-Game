Mini-Python-Arrow-Control-Game
==============================

Minigame where you are an 'X' and you eat 'O's  



COMPATIBLITY:

Python 2.7 and 3.3 availible!

Launch form terminal/command prompt with "python Game27.py" or "python Game33.py"

GAME:

Get 20 points to win

Use WASD to move
